New Order Pop-Up Plugin is an alert that will notify you of any new orders that you receive in your WP Admin area.

== Description ==
New Order Pop-Up Plugin is perfect when you need to do a quick fulfilment.

This plugin will:

1. Create an alert with sound notification for every new order
2. You can customize everything in the alert: text, colours and sound notification.

Plugin features of the free version:
- Default alert with no sound

Plugin features of the Pro version:
- Customize text and colours.
- Add your own notification sound
- Upload your own icon. 

For more information visit http://mozzoplugins.com/.
